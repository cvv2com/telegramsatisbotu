# Gift Card Images

Place your gift card images in this directory with the following naming convention:

## MC Picture Cards
- Front: mc{ID}front.jpg (e.g., mc1front.jpg)
- Back: mc{ID}back.jpg (e.g., mc1back.jpg)

## Visa Picture Cards
- Front: visa{ID}front.jpg (e.g., visa1front.jpg)
- Back: visa{ID}back.jpg (e.g., visa1back.jpg)

Make sure images are in JPG format and contain:
- Front: 16-digit card number and expiration date
- Back: 3-digit PIN code
